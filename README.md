# Hi, I'm Gavin 👋

I'm a data scientist and civic technologist based in New Jersey. I spend a lot of time thinking about how public data can be made more useful, more accessible, and more honest — and building things to test that theory.

My work sits at the intersection of government, data infrastructure, and open source software. I've built tools for journalists, researchers, attorneys, and curious citizens who want to understand how their communities work.

---

## 🗂️ Notable Projects

| Project | What it does |
|---|---|
| [**zipcodeR**](https://github.com/gavinrozzi/zipcodeR) | R package for working with U.S. ZIP codes — geocoding, demographics, spatial joins. [Published in *Software Impacts*](https://doi.org/10.1016/j.simpa.2021.100099). |
| [**njgeo**](https://github.com/gavinrozzi/njgeo) | Geocode NJ addresses using the state's official service. No API key needed. |
| [**njtr1**](https://github.com/gavinrozzi/njtr1) | R interface to NJ traffic crash data (NJTR-1 reports). |
| [**1033-watch**](https://github.com/gavinrozzi/1033-watch) | Dashboard tracking military equipment transfers to U.S. law enforcement under the 1033 program. |
| [**OPRAmachine**](https://opramachine.com) | Public records request platform for New Jersey (founded; since transferred). |
| [**NJCSNavigator**](https://njcsnavigator.com) | Tool for navigating New Jersey civil service job titles and compensation data. |
